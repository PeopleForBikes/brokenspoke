Shortcodes

A tool to convert a City Ratings CSV file to a Shortcode CSV file.

Basically, the 2 files will be identical, except that the shortcode file headers will be at most 5 character longs, without underscores nor white spaces.

Goal

The goal is to simplify the templates that will be used when performing the data-merge operation. If the templates had to deal with the full names, it would be more difficult for the designers to put them together.
